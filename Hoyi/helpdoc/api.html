    1. 使用方式
         <%%# 和 #%%> 包含函数。函数调用方法，this 或者 modelInfo 调用,
         this 表示当前对象，modelInfo表示一个全局的对象，对象内容请见：
         Fields 操作函数 请点这里
         Column 类 请点这里
         model 类 请点这里
         如:
         <%%# modelInfo.NameSpace #%%>
         <%%# this.EntityName #%%>
        

         点击查看Column 中的使用方法         

         this 表示当前对象，modelInfo表示一个全局的对象.
         <%%# 这个范围内可以使用this 和 modelInfo 做关键词 #%%>

         <%%# [Fields[inner,0,last(1)]={ <%#= this.TypeName.tojavapackclass() #%> <%#= this.ColumnName #%> , }] #%%>

         <%%# [Fields[last(1)]={ <%#= 这里的this， #%> <%#= this.ColumnName #%> , }] #%%>
